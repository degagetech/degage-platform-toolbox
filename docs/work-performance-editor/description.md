## 工作绩效编辑器（WorkPerformanceEditor) ##

支持的操作系统： Windows ![windows](../windows.png)

IDE：VS2015/17/19

​       这个工具用于来获取、编写、发送那每月需要上交的绩效考核文件。

您可以通过此工具：

- 通过模板文件以适配不同绩效考核文件格式。
- 获取您工作邮箱中与绩效相关的邮件，并阅读与修改附件。
- 将您处理后的绩效文件以及邮件的形式发送给指定或多个收信人。
